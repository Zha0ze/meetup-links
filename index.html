01 Where the work happens
    Claude Code →
    The coding agent I mostly work in. Build and refine your site in the terminal.
    Claude.ai / Cowork →
    Prepare docs, refine your CV, positioning and case studies — before any code.
    Claude Design →
    Very visual person? Start here — tweak & comment are built in — then move to Claude Code.
    Paper →
    A canvas that is the DOM — HTML/CSS native, so the agent goes design ↔ code easily. Where I explored my layout.
    Vercel →
    Free Hobby hosting + password protection via env variable. You only pay for a domain.
02 Give the agent tools
    Playwright MCP →
    Browser control, so the agent can check its own work in a real browser.
    agent-browser · Vercel Labs →
    Another way to let the agent walk a live page.
    Notion MCP →
    Reach notes and docs that live in Notion.
03 Craft tools
    Toolcraft · Pixel Point →
    Bring the library in, describe your tool, and the tweak-UI almost builds itself — even WebGL.
    DialKit · Josh Puckett →
    Slider "bars" to tune styles, animation and layout by eye. See his Interface Craft course.
    Agentation · Benji Taylor →
    Comment on your live UI like a design review, then send the whole batch of edits to any agent.
    Refine · Jakub Antalik →
    Tune UI transitions live on a timeline while building, then save back to code.
04 Skills for your agent
    Skills.sh · Vercel →
    Open, curated library of agent skills. Install into Claude Code, Cursor, Copilot…
    UI Skills →
    Skills for design engineers — opinionated constraints so agent-built UI stays consistent and accessible.
    Rams →
    Free design review right inside your agent — 8-category checklist (a11y, layout, type, contrast…).
    Emil Kowalski — 3 skills →
    npx skills add emilkowalski/skills
    • emil-design-eng — his philosophy on UI polish, animation and the invisible details.
    • review-animations — strict review of your animations against his craft bar.
    • animation-vocabulary — turns vague prompts into the right motion terms.
    jakubkrehel/skills →
    Product-design skills: animations, typography, layout, colors.
    npx skills add jakubkrehel/skills
    case-study skill · Jakub Šalmík →
    Structure and storytelling for portfolio case studies. (On his GitHub, @scoobynko.)
    Chat with Miranda →
    Check your portfolio and case studies against a hiring-manager lens.
